## Base URL

```
https://sadi.digifarma.com.br/api/
```

## Autenticação

Todas as rotas (exceto `GetToken` e `SetSenha`) exigem os headers:

| Header | Descrição |
| --- | --- |
| `x-digifarma-user` | Usuário emitido pela Digifarma |
| `x-digifarma-token` | Token obtido via `GetToken` |

Veja [Autenticação](/docs/authentication) para o fluxo completo e os pré-requisitos de credenciais.

## Como usar essa referência

- **Menu à esquerda** — endpoints agrupados por área (Autenticação, Loja, Cliente, Produto, Vendas, Utilitários, SADI Online)
- **Try It** — clica no endpoint, preenche os campos, dispara a chamada direto do browser
- **Exemplos de código** — a doc mostra o request em curl, Python, JavaScript e outras linguagens
